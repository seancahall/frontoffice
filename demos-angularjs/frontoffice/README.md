# LeadEssentials FrontOffice

### Requirements
* NodeJS >= 8.x
* Npm >= 5.6

### Development
1. Copy `.env.example` to `.env` and set parameters if needed.
2. Install dependencies.
```sh
npm install
```

3. Run development environment.
```sh
npm run dev
```

### Build
1. Copy `.env.example` to `.env` and set parameters if needed.
2. Run build process
```sh
npm run build
```
